前端面试题基础篇（三）

发表于 2022-07-28

消除浮动有哪些方法？

  1. 触发BFC
  2. 多创建一个盒子，添加样式：clear：both
  3. after方法

position有哪些值？分别是根据什么定位的？

static（默认） 没有定位

fixed 固定定位，相对于浏览器窗口进行定位

relative 相对于自身定位，不脱离文档流

absolute 相对于第一个有relative的父元素，脱离文档流

relative和absolute区别：

  1. relative不脱离文档流、absolute脱离文档流
  2. relative相对于自身、absolute相对于第一个有relative的父元素
  3. relative有left、top值
  4. absolute有left、right、top、bottom值

css sprite是什么 有什么优缺点？

sprite：将一个页面涉及到的所有零星图片都包含到一张大图中去。

优点：减少了http请求的次数，提高了性能。

缺点：维护比较差（例如图片位置进行修改或者内容宽高修改）

display:none 和visibility:hidden的区别？

1.作用不同：

​ visibility:hidden 将元素隐藏，但是仍占用位置

​ display:none 不占用位置

2.重绘和回流的问题：

​ visibility:hidden 产生重绘

​ display:none 产生重绘和回流

​ 产生回流一定会造成重绘，但是重绘不一定会造成回流。

产生回流的情况：改变元素的位置（left、top…）、显示隐藏元素…

产生重绘的情况：样式改变（background、color …)

opacity和rgba区别

共同性：

  1. opacity 取值范围0到1之间，0表示完全透明，1表示不透明
  2. rgba R表示红色，G表示绿色，B表示蓝色，取值可以在正整数或者百分数。A表示透明度取值0到1之间

区别：

​ opacity会继承父元素的opacity属性，而RGBA设置的元素的后代元素不会继承不透明属性。
