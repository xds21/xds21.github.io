js数组和对象的遍历方式

  1. js遍历数组
    • for循环
    • for… in 遍历（当前对象及其原型上的）每一个key 。以任意顺序遍历对象的可枚举属性，（最好不要用来遍历数组）因此当迭代哪些访问次序重要的arrays是用整数索引去进行for循环（或者使用Array.prototype.forEach()或for…of循环）
    • for…of 遍历（当前对象及其原型上的）每一个value。（ES6）允许你遍历Arrays（数组），String（字符串），Maps（映射），Sets（集合）等可迭代的数据结构等。不能遍历普通对象
    • forEach 在遍历过程中不能被终止，必须每一个值遍历一遍后才能停下来
    • map
  1. js遍历对象
    • for…in
    • Object.keys 「注意」for…of不能单独来遍历对象，要结合Object.keys一起使用才行。 Object.key()方法会返回一个由一个指定对象的自身可枚举属性组成的数组，数组中属性名的排列顺序和使用for…in循环遍历改对象时返回的顺序一致。
    • Object.getOwnPropertyNames(obj) 返回一个数组，包含对象本身的所有属性（包括不可枚举属性）遍历可以获取key和value

map和forEach的区别

  • forEach 方法，是最基本的方法，就是遍历和循环，默认有3个参数：分别是遍历的数组内容item、数组索引index，和当前遍历数组 Array
  • map方法，基本用法与forEach一致，但是不同的，它会返回一个新的数组，所以callback需要有 return 值，如果没有会返回 undefined

跨域问题

同源策略：同源指的是域名、协议、端口号相同

如何解决跨域

  • jsobp跨域
  • document.domain + iframe 跨域
  • nodejs中间件代理跨域
  • 后端在头部信息里设置安全域名

attribute和property的区别是什么

  • attribute 是 dom 元素在文档中作为 html 标签拥有的属性
  • property 就是 dom 元素在 js 中作为对象拥有的属性
  • 对于 html 的标准属性来说，attribute 和 propwety 是相同的，是会同步更新的
  • 当对于自定义的属性来说，他们是不同步的
